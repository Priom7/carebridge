# Emergency Help Journey

Status: Implemented locally in UX-009

1. A parent taps **I need help** on a due medicine reminder.
2. The reminder becomes escalated and an active emergency alert is created.
3. Priority 1 contacts with emergency permission are notified.
4. A local contact opens the alert and taps **I am going to help**.
5. Caregivers see the accepting contact and a chronological timeline.
6. The caregiver resolves the alert after confirming the parent is safe.

This workflow supports coordination only. It does not replace local emergency services or guarantee delivery.
