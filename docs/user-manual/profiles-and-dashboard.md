# Care Profiles and Dashboard

## Switch between people

Use the profile control in the top-right header. Select an active profile such as Father or Mother. The dashboard changes its metrics, timezone, appointment, and actions to that person.

Always confirm the selected profile before adding medicines, reports, contacts, or health information.

## Add a care profile

1. Open **More → Care profiles**.
2. Select **Add care profile**.
3. Enter the person’s name, preferred name, and relationship.
4. Add contact, location, timezone, and language details.
5. Add health conditions, allergies, mobility notes, doctor notes, and emergency instructions.
6. Confirm the primary and secondary caregiver.
7. Select **Save profile**.

CareBridge stores information for organization and coordination. It does not diagnose conditions or replace medical advice.

## Edit or complete a profile

Select **Edit care profile** from the profile list or use the edit button on the dashboard. A dashboard banner identifies profiles missing important contact, timezone, or emergency information.

## Archive a profile

From Care profiles, select the archive icon and confirm. Archived profiles disappear from active switching but remain listed separately so their history is not silently lost.

## Read the dashboard

The selected person’s dashboard shows:

- Medicines due
- Items needing attention
- Documents added this week
- Next appointment
- Quick care actions

The current UI uses realistic demonstration data. Medicine, document, emergency, and appointment quick actions become complete workflows in their scheduled UI tasks.
