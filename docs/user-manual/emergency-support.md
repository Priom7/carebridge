# Emergency Support

## Set up contacts

Open **More → Emergency contacts → Add contact**. Set Priority 1 for people most able to respond nearby. Grant only the information they need; document and location access are off by default.

## Request and coordinate help

A parent can tap **I need help** from a medicine reminder. Caregivers can open **More → Emergency help alerts**, notify Priority 1 contacts again, record who is going to help, and resolve the alert.

## Important limitation

CareBridge cannot guarantee notifications, calls, or WhatsApp delivery. If someone may be in immediate danger, contact the appropriate local emergency service directly.
